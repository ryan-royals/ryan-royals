## Azure Data Factory

Azure Data Factory is a cloud based Extract-Transform-Load and Extract-Load-Transform tool used to manage data within Azure

It consists of the following key components:

- Pipelines
- Activities
- Datasets
- Linked Services
- Data Flows
- Integration Runtimes
