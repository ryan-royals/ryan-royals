Azure Private Link is the overarching service used to manage and maintain [[90_slipbox/Azure Private Endpoint\|Azure Private Endpoint]] and [[90_slipbox/Azure Service Endpoint\|Azure Service Endpoint]].

Private Links can be managed in the **Private Link Centre** in the Azure Portal.
