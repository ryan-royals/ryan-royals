Azure synapse is a enterprise analytics tool for managing multiple data warehouses and big data systems.  
Key components are:

- SQL
- Apache Spark
- Data Explorer
- Pipelines

Synapse is built upon [[90_slipbox/Azure Data Factory\|Azure Data Factory]] and as such shares many features.
