Using a zk_prefix on files in [[90_slipbox/Obsidian\|Obsidian]] just seems to add more friction to creating notes, as it feels like a rabbit hole of spending time to hide the ID, and it is by the end of it seemingly doing nothing.

I like the idea of having a guid as then I dont have to consider the file name, but I would want to spend so much time eventually just hiding that guid that it feels fruitless.  
The only time my notes have clashing names is for meetings, which get a date on them anyway.
