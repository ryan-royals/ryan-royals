Trivy is a [[90_slipbox/Terraform\|Terraform]] tool used to scan for vulnerabilities and misconfigurations, such as Access Controls being disabled on Azure Storage Accounts.

Options for bypassing these rules can be found: [Filtering - Trivy (aquasecurity.github.io)](https://aquasecurity.github.io/trivy/v0.48/docs/configuration/filtering/#by-inline-comments)
