## Summary

This article explains how to create, modify, or delete public IP address prefixes in Azure. You can assign a static public IP from a prefix to Azure resources. Use CLI or PowerShell commands to manage public IP address prefixes efficiently.

## Highlights

A public IP address prefix is a contiguous range of standard SKU public IP addresses. When you create a public IP address resource, you can assign a static public IP from the prefix and associate the address to Azure resources.
